# MineStream: Video Calling App for Android and iOS

## Features

+ Google Sign In
+ Create New Meeting
+ Join Meeting
+ Mute/Unmute Audio
+ Mute/Unmute Video
+ Change Display Name while Joining
+ Raise Hand
+ Start Screen Sharing
+ Mute Everyone
+ Start Live Stream
+ Screen Share
+ Chat
+ Switch Front/Rear Camera
+ Picture in Picture
+ See Meeting History
+ Log out
